Developer Tip
The Electron framework lets you write cross-platform desktop applications using JavaScript, HTML and CSS. It is based on Node.js and Chromium.Client side frameworks like Angular.js or React.js can also be used with Electron to leverage Single page application to become a desktop application.Electron make it possible to achieve performance like a desktop application and rich UI like web application.The platforms supported by Electrons are OS X,Linux and Windows. Some of the popular applications developed using Electron are Atom editor,Microsoft Visual Studio Code and Slack.
  1. Home Page : http://electron.atom.io
  2. Github Repo: https://github.com/atom/electron
  3. Useful Resources: https://github.com/sindresorhus/awesome-electron
