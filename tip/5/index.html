Developer Tip

Careful while working with big numbers in javascript. By default all numbers in javascript are floating point (sign * mantissa * 2^exponent where mantissa is 53bits). Hence the max integer supported in JS is 2^53-1 = 9007199254740992.

Example: Twitter JSON: 

{ id: 10765432100123456789,id_str : "10765432100123456789", ...}
id in Twitter are 64 bit long .
> parseInt("10765432100123456789")
10765432100123458000

As you can see above the precision is lost with large numbers. Languages like C/C++/Java supports 64bit unsigned integers.
