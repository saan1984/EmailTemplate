React Radium

Radium is collection of tool for better management of styles in React elements. Some of the key features of Radium are as follows:-

  1. simple inline styles.
  2. supports pseudo class like :hover, :focus and :active etc.
  3. Media queries for responsive design.
  4. Automatic vendor prefixing like -moz, -webkit, -o and -ms to support experimental or nonstandard CSS properties.
  5. Supports CSS3 Keyframes animation.
  6. ES2015 class and createClass support.

Example:-

Following code shows the basic use of Radium , it creates a Radium wrapper around a React component There are many useful feature for managing style in React. 

  
      export default radium(MyReactComponent);
