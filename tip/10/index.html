How to hook during DLL loading and get the complete call stack quickly?

It helps to identify which module is loading a specific dll and see the complete call stack quickly.

  1. Add HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\msxml3.dll". (note: add more keys like this for the dll you would like to hook)
  2. Add a DWORD value under this key with name BreakOnDllLoad and set its value as one.
  3. Run your application in VS in debug mode (directly run from VS than attaching it)
  4. Add a new break point ('Break at function' menu in VS) with below entry. And invoke your app in debug mode in VS {,,msxml3.dll}DllMain
